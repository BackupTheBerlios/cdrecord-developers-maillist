If you read the man page for cdrecord, you will find that dev=/dev/cdrom 
and similar are unsupported.

Just use the offcial dev=b,t,l syntax and add an "emergency" method
to allow manual entering of exceptional dev= parameters. This exmergency 
method is currently only needed on Mac OS X and AIX.
