Be careful, this does not work out of the box.....

Microsoft delivers rotten include files with their compiler.

Once you have a POSIX compliant filesystem (and MS switches HTFS into such a 
mode when inside SFU), the MS compiler is no longer able to read it's own 
include files as these include files use random upper-/lower-case characters 
when including their own files and that random strings do not match the real 
file names.
